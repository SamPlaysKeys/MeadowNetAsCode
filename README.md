# MeadowNetAsCode

Welcome to **MeadowNetAsCode**. This project is a documentation and automation of my home lab infrastructure, currently a work in progress as I migrate everything into this public repository.

## What’s going on here?
I’m in the middle of a transition: moving away from a local Ansible Vault file toward a more robust setup using the free version of [Hashicorp Vault](https://www.hashicorp.com/products/vault). 

This repository isn't just about keeping my home server running; it’s an experiment in "working in public." By moving my infrastructure-as-code here, I’m hoping to explore and improve my workflows while allowing others to see (and maybe learn from) my mistakes and iterations. 

This follows the same spirit as my other public projects, like my [dotfiles](https://github.com/SamPlaysKeys/dotfiles), which serve as a playground for my non-professional explorations and technical curiosities.

## Project Scope
While this repo showcases many of the public-facing elements of my infrastructure, it is **not** a complete map of every project I’m running. Instead, it’s designed to be a self-contained lab environment that handles idempotent deployments across my home network.

## A Note on AI & Implementation
You’ll see some documentation and notes here that were polished with the help of AI to keep things readable. However, the **actual implementation**—the logic, the architecture, and the configurations—is entirely my own work.

## Life Update & Pace
Just a heads-up: progress might be a bit slow through August 2026. I’m currently in the process of buying a new house, so my focus is split between digital architecture and physical walls for a while.
